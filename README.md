# processing-picross3d

A program for illustrating 3D Picross instances.

## Usage
* Drag: Rotate the model.
* Shift + Drag up/down: Zoom in/out.
* Ctrl + Click: Erase a cube.
* Press 'u': Undo.
* Press 'r': Reset the model.
* Press 'a': Show the answer.
