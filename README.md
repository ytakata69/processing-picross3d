# processing-picross3d

A program for illustrating 3D Picross instances.

## Usage
* Drag: change the camera position.
* Drag up/down + shift: zoom in/out.
* Click + control: erase a cube.
* Press 'u': undo.
* Press 'r': reset.
* Press 'a': show the answer.
